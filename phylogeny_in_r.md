
Neighbor Joining and UPGMA
We can estimate trees from distance matrices using neighbor-joining and UPGMA algorithims


```r
fasta_UPGMA <- upgma(dna_dist)
fasta_NJ  <- NJ(dna_dist)
plot(fasta_UPGMA, main="UPGMA")
```
